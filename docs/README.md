# 文档目录

此目录存放项目文档。

## 文档结构

```
docs/
├── architecture/          # 架构文档
│   ├── overview.md        # 系统架构概述
│   └── database.md        # 数据库设计
├── api/                   # API 文档
│   └── openapi.yaml       # OpenAPI 规范
├── ai/                    # AI 相关文档
│   ├── prompts.md         # Prompt 模板说明
│   └── schemas.md         # JSON Schema 定义
└── deployment/            # 部署文档
    └── docker.md          # Docker 部署指南
```

## 文档规范

- 使用 Markdown 格式
- API 文档使用 OpenAPI 3.0 规范
- 保持文档与代码同步更新
